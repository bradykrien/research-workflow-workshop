# Accessibility Statement
{: .fs-8 }

---

## Our commitment

This workshop site is designed to be accessible to the widest possible audience, including people who use assistive technologies. We aim to conform to the [Web Content Accessibility Guidelines (WCAG) 2.1](https://www.w3.org/TR/WCAG21/) at Level AA.

## Features of this site

- **Semantic HTML structure:** Pages use proper heading hierarchy, landmarks, and HTML elements to support screen readers and keyboard navigation.
- **Keyboard navigation:** All content and links are reachable via keyboard. The site theme includes a skip-to-content link and supports standard tab navigation.
- **Text-based content:** Guides are written in plain text and Markdown, avoiding reliance on images or visual-only cues to convey information.
- **Readable typography:** The site uses a legible font stack with adequate size and spacing. Text reflows at different screen sizes.
- **Descriptive link text:** Links describe their destination rather than using generic phrases like "click here."
- **Sufficient color contrast:** The site theme meets WCAG AA contrast requirements for text and interactive elements.

## Known limitations

- **External tools:** The software tools described on this site (Zotero, Tropy, BrowZine, Mendeley, Google Scholar) have their own accessibility profiles, which vary. We encourage you to consult each tool's documentation for information about their accessibility support.
- **External links:** We link to third-party documentation and resources whose accessibility is outside our control.
